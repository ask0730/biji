# Python脚本打包说明

## 方法一：使用打包脚本（推荐）

1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

2. 运行打包脚本：
   ```bash
   python 打包成exe.py
   ```

3. 打包完成后，exe文件会在 `dist` 文件夹中

## 方法二：手动使用PyInstaller

### 安装PyInstaller
```bash
pip install pyinstaller
```

### 打包单个文件

**文件整理.py:**
```bash
pyinstaller --onefile --noconsole --name 文件整理 文件整理.py
```

**万方数据.py:**
```bash
pyinstaller --onefile --noconsole --name 万方数据 万方数据.py
```

**知网数据.py:**
```bash
pyinstaller --onefile --noconsole --name 知网数据 知网数据.py
```

## 注意事项

1. **配置文件**: 
   - `config.txt` 需要和exe文件放在同一目录
   - 或者放在exe文件所在目录

2. **Chrome驱动**:
   - 万方数据和知网数据需要Chrome浏览器和ChromeDriver
   - 确保系统已安装Chrome浏览器
   - ChromeDriver会自动下载（如果使用webdriver-manager）

3. **输出文件**:
   - 打包后的exe文件在 `dist` 文件夹中
   - 临时文件在 `build` 文件夹中（可以删除）

4. **文件大小**:
   - 由于包含selenium等库，exe文件会比较大（约50-100MB）
   - 这是正常现象

## 打包参数说明

- `--onefile`: 打包成单个exe文件
- `--noconsole`: 不显示控制台窗口（GUI模式）
- `--name`: 指定输出文件名
- `--clean`: 清理临时文件
- `--distpath`: 指定输出目录
- `--workpath`: 指定临时文件目录

## 常见问题

1. **打包失败**: 检查是否安装了所有依赖
2. **exe无法运行**: 检查是否有杀毒软件拦截
3. **缺少模块**: 确保所有import的模块都已安装
